Welcome to game recommender app

All game searches are stored to a DB in case you wish to keep track of the games you want to play.

In case you want to retrieve them, access to endpoint "/dump_db"

Endpoints:

* "/get_random" in order to get first games from the Query Response.

* "/get_genre/?genre=GENRE_ID" in order to only retrieve games from a certain genre. Please check for the list of supported genres at the bottom of the page

* "/get_info/?game_id=ID" get info from a defined game.

* "/dump_db/" get info from all games stored in db.
